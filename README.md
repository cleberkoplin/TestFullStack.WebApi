# TestFullStack.WebApi
WebApi architeture developed to serve as backend with endpoints controllers, services, database connection and automated test.

Architecture:
- AspNetCore 2.1.1
- NETCore 2.1
- MVC
- Entity Framework Core
- JWT
- SimpleInjector
- Services
- Cryptography HashMD5

Test
- XUnit 2.3.1
- NetTest 15.8.0
